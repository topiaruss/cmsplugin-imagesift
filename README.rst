A Gallery that dynamically selects images from imagestore.

Documentation
-------------

The full documentation is at https://cmsplugin-imagesift.readthedocs.org.

Quickstart
----------

Install cmsplugin-imagesift::

    pip install cmsplugin-imagesift

Then use it in a project::

    import cmsplugin-imagesift
